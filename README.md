# Beargit - clone of 'Git' version control system (CS61C Project)

### Commands supported

1. beargit init
2. beargit add <file>
3. beargit commit -m <message>
4. beargit log
5. beargit checkout <commit>
6. beargit checkout <branch>
7. beargit checkout -b <new_branch>

### Build

1. source init_test

### Testing

1. make beargit-unittest
2. source init_test
3. beargit-unittest
