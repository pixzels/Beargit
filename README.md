# Beargit - Clone of 'Git' version control system

### Commands supported

1. beargit init
2. beargit add `file`
3. beargit rm `file`
4. beargit commit -m `message`
5. beargit log
6. beargit status
7. beargit branch
8. beargit checkout `commit`
9. beargit checkout `branch`
10. beargit checkout -b `new_branch`

### Build

1. source init_test

### Testing

1. make beargit-unittest
2. source init_test
3. beargit-unittest
